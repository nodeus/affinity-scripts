# Affinity Scripts

Collection of scripts for Affinity by Canva, managed via MCP server.

## Scripts

| Script | Description | Version |
|--------|-------------|---------|
| [affinity-color-palette-gen](https://github.com/nodeus/affinity-color-palette-gen) | Extract fills, strokes, gradients into organized swatches | 9.1.1 |
| [affinity-hanging-chars-and-prepositions](https://github.com/nodeus/affinity-hanging-chars-and-prepositions) | Fix orphaned characters and prepositions with non-breaking spaces | 1.0.0 |
| [affinity-chart-builder](https://github.com/nodeus/affinity-scripts/tree/main/affinity-chart-builder) | Build bar, column, and pie diagrams based on the data in the text object directly in affinity | 1.3.0 |
