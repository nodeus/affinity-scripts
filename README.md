# Affinity Scripts

Collection of scripts for Affinity by Canva, managed via MCP server.

## Scripts

| Script | Description | Version |
|--------|-------------|---------|
| [affinity color-palette-gen](affinity%20color-palette-gen/) | Extract fills, strokes, gradients into organized swatches | 9.1.0 |
| [affinity hanging-chars-and-prepositions](affinity%20hanging-chars-and-prepositions/) | Fix orphaned characters and prepositions with non-breaking spaces | 1.0.0 |
